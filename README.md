# Text Classification using TF-IDF and Logistic Regression

## Overview
This project focuses on **text classification** using traditional Natural Language Processing (NLP) and Machine Learning techniques.  
The pipeline includes **text preprocessing with NLTK**, **feature extraction using TF-IDF**, and **classification using Logistic Regression**.

The goal of this project is to demonstrate an end-to-end NLP workflow, from raw text preprocessing to model training and evaluation.

---

## Tech Stack
- Python
- NLTK (text preprocessing)
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

---

## Workflow
1. Text cleaning and preprocessing using **NLTK**
   - Lowercasing
   - Tokenization
   - Stopword removal
   - Lemmatization / Stemming
2. Feature extraction using **TF-IDF**
3. Model training using **Logistic Regression**
4. Model evaluation and prediction
